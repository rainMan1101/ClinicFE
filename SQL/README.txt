Создание дампа:
	pg_dump имя_БД > файл_дампа
	
Создание базы данных из дампа:
	CREATE DATABASE clinic_db ENCODING = 'UNICODE';
	\q 
		- выход
	psql имя_БД < файл_дампа	
	
	
Создание БД при помощи скриптов:
	1) \! pwd 
		- просмотр текущего каталога
	2) \cd <папка_со_скриптами>
		- переход в нужную папку
	3) \i 'make_all.sql'
		- выполнить файл 'make_all.sql' (создать базу данных)
		
Удаление БД при помощи скриптов:
	1 - 2
	3) \i 'drop_all.sql'
		- выполнить файл 'drop_all.sql' (удалить базу данных)
